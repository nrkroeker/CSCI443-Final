Project Proposal

For this project, David Haines and Nathalie Kroeker will work together to design and implement a database managing data for an animal shelter. The information stored in the database could include important facts about animals currently residing at the shelter, description and attendance at meet and greet adoption events, general employee data, foster parents, and/or adoption records. To expand the project further, the tasks and operations of a shelter could be described more specifically in order to add detail and complexity to the database.

Creating the database will integrate many skills learned in class including building a schema, drawing descriptive ER diagrams, normalization, and writing SQL queries to access or alter data. Additional facets of our project will include researching the best ways to connect a database to a program, creating a command line interface to accept user input, and potentially creating a more in-depth GUI displaying or altering records in the database. When working on the project through the rest of the semester, we will construct the relations and schema together. During normalization, we will likely each review the whole schema individually, and then combine our individual comments and ideas. David will do more research into how database connections work in different programming languages, while Nathalie will look more into integrating the database records into a website or GUI. We will write the code and any necessary setup queries in SQL together.

We will be most technically challenged in this project when integrating the database with an actual program, because neither of us has much experience with that task. In addition, we want to ensure that we properly normalize and refine the database. So, the challenge of applying general concepts from class into a specific scenario with many different relations will take additional effort. When the project is complete, we can demonstrate the efficacy of the database by walking through the process of an animal's data and records from arrival at the shelter to adoption.
